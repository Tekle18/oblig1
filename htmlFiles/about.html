Project report

At the start of the project we discussed how the project is going to be structured and the layout of the pages. At the end of this meeting we created a rough guiding line for the project structure and page layout. One of the important decisions were that we were going to use a css file for each html file that we have. This was dicided so that we can have clearer code and better project structure. We also decided to use a made up persona for the project. The guidelines for the page layout we made at the start include to use as many photos as possible and to try to align these with some texts like description and list to hold hobbies. At the end we can say that we followed our guidlines and ended up making a web site that works and shows the persons ability and achivments. This is offcourse besides the minor issues that we faced which will be discussed on each pages report.

Homepage

The home page is designed using two images. This images were styled using the tag <div>. The reason we chose to do it this way is mainly to align the different elements we used on the same line. We managed to create the navigation bar which is carrying the links to the different pages of site. These works like they suppose to and the are made using the <ul> and <il> tags. The page might be exepirienced as empty but we struggled to have lists underneath. The reason waas when we try to make other lists that can contain hobbies for example, they we ruined by the list we defined above for the links of the pages.
